# SmartAppointment

Fullstack Appointment Management System

---

## 🚀 Tech Stack

### Backend
- ASP.NET Core Web API
- Entity Framework Core
- PostgreSQL
- FluentValidation

### Frontend
- React (Vite)
- Axios
- JavaScript (ES6)

---

## 📌 Features

- Create appointment
- List appointments
- Delete appointment
- Backend API integration
- RESTful architecture
